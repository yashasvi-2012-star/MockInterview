VIPER-VIPER

A SIGNAL. A GRID.
NO ROOM FOR MISTAKES.
VIPER-VIPER is a classic snake game that i made to learn coding games . Guide the signal across a 20×20 grid, consume data nodes to grow longer, and survive for as long as you can — every bite makes the run faster and less forgiving.
FEATURES
LIVE SCORING Real-time score and a running best-of-session high score, always in view.
RISING SPEED The signal accelerates with every node eaten, so the challenge never stalls.
RESPONSIVE CONTROLS Rapid direction changes are queued instantly, so tight escapes always register.
PLAYS ANYWHERE Keyboard, on-screen D-pad, or swipe gestures — desktop and mobile alike.
